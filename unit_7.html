Unit 7: Secure Coding Practice

Collaborative Discussion 3

Reflection

Unit 7 focused on secure coding practice.

At this point my understanding changed. Secure code is not only code that works, but code that also behaves safely when it is misused.

The authentication task demonstrated this clearly. The original system stored passwords in plaintext, accepted weak passwords, did not validate input, did not limit failed attempts and did not log authentication activity.

This unit helped me understand security as something that should be designed into the system early, rather than added afterwards.
